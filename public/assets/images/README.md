## No es obligatorio subir todo de una vez

El sitio funciona igual sin estas imágenes (usa marcadores de posición
turquesa con el texto descriptivo). Puedes ir subiendo por carpetas
—por ejemplo, primero `doctora/` y `clinica/`, que son las de mayor
impacto— y las voy conectando conforme lleguen.
